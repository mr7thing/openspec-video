# Standard Directory Structure

This document details the file structure created by `opsv init` for every new video project.

## Root Directory
- **`videospec/`**: (Core) Contains all creative assets and scripts.
- **`.antigravity/`**: (System) Agent workflows and rules.
- **`queue/`**: (Generated) Job files for the Agent.
- **`artifacts/`**: (Generated) Final images and videos.

## `videospec/` Breakdown
This is where the user works.

### 1. `project.md`
The project manifesto.
```markdown
---
title: "My Movie"
style: "Cyberpunk"
resolution: "3840x2160"
---
# Context
A dark noir detective story...
```

### 2. `assets/` (Markdown Assets)
- **`characters/*.md`**: Actor definitions.
  ```markdown
  ---
  id: "char_k"
  name: "Detective K"
  ---
  ![Reference](./k_ref.png)
  ```
- **`scenes/*.md`**: Set definitions.

### 3. `stories/`
- **`Script.md`**: The screenplay.
  ```markdown
  **Shot 1**: [char_k] enters [loc_bar].
  ```

### 4. `changes/`
- Created by `opsv proposal`. Tracks major creative decisions.
