## 一、引用语法进化

### 1.1 注解式引用 (Annotative Referencing)
为了保持剧本的文学可读性，推荐使用括号将资产 ID 作为注解嵌入：
- **语法**：`Hero (@hero) 从 废墟 (@ruins) 站起来。`
- **优点**：括号外可使用任何昵称，括号内精准锁定资产。
- **解析行为**：系统会自动识别 `(@id)` 或 `@id`，在生成 Prompt 时将括号内的 ID 替换为对应的视觉属性。

### 1.2 镜头局部参考 (Shot-Local References)
对于仅针对单个镜头的视觉参考（无需全局建模的素材），可直接在分镜主体中插入图片：
- **用法**：在 `Script.md` 的 `## Shot NN` 章节下直接使用 `![描述](路径)`。
- **逻辑**：`opsv generate` 会自动将该分镜下所有本地图片归集为该 Job 的参考图。
