# Project Context: Cyberpunk Detective Demo

## 1. Core Narrative
**Logline**: A cybernetic detective uncovers a digital phantom in the ruins of Old Tokyo.
**Theme**: Memory vs Data, Soul vs Silicon.
**Tone**: Gritty, Atmospheric, Melancholic but slick.

## 2. Visual Style
- **Genre**: Cyberpunk / Neo-Noir.
- **Palette**: Deep blacks, neon cyan (#00FFFF), crimson red (#FF0033), rain-slicked wet streets.
- **Lighting**: Volumetric fog, Chiaroscuro (high contrast), practical light sources (neon signs).
- **Aspect Ratio**: 2:39:1 (Anamorphic Cinema).
- **Resolution**: 4K UHD.

## 3. Technical Constraints
- **Model**: Nano Banana Pro (Gemini 3 Pro Image).
- **Character Consistency**: STRICT. Must use Reference Sheet `assets/characters/K.yaml`.
- **Text Rendering**: High (Japanese Katakana support required).
- **Motion**: Cinematic slow-motion (Veo 3.1).

## 4. Key Characters
### Detective K
- **Role**: Protagonist.
- **Trait**: Stoic, wears a worn leather trench coat, right eye is robotic (red lens).
- **Ref**: `assets/characters/detective_k.png`

## 5. Key Scene
### The Void Bar
- **Description**: A cramped ramen bar floating in zero-g.
- **Atmosphere**: Smoky, yellow lanterns, holographic menu.
