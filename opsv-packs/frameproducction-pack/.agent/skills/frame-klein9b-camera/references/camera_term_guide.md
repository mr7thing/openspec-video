# Camera Term Guide — 镜头术语完整列表

## 1. 机位角度

| 术语 | 说明 |
|------|------|
| `frontal shot` | 正面镜头 |
| `extreme low angle shot from the front` | 极低仰角正面 |
| `low angle shot from 45 degrees to the right` | 右下45度仰角 |
| `low angle shot from 45 degrees to the left` | 左下45度仰角 |
| `ultra-low angle upward shot at 30 degrees from the right` | 右30度超低仰角 |
| `ultra-low angle upward shot at 30 degrees from the left` | 左30度超低仰角 |
| `ultra-low angle upward shot at 135 degrees from the right` | 右135度超低仰角 |
| `ultra-low angle upward shot at 135 degrees from the left` | 左135度超低仰角 |
| `low angle back shot looking up` | 低角度背影仰视 |
| `45-degree overhead shot from the right` | 右45度俯拍 |
| `45-degree overhead shot from the left` | 左45度俯拍 |
| `135-degree high angle downward shot from the right side` | 右135度高位俯拍 |
| `135-degree high angle downward shot from the left` | 左135度高位俯拍 |
| `high-angle downward shot from the back` | 高位背影俯拍 |
| `high-angle shot` | 高角度镜头 |
| `top-down shot` | 顶视图 |
| `bird's-eye view` | 鸟瞰视角 |

## 2. 镜头运动

| 术语 | 说明 |
|------|------|
| `The lens rises to waist height` | 镜头升至腰部高度 |
| `The lens rises to chest height` | 镜头升至胸部高度 |
| `The lens descends to knee height` | 镜头降至膝盖高度 |
| `The camera pans down to the ground` | 镜头下摇至地面 |
| `camera horizontal translation left` | 水平左移 |
| `camera horizontal translation right` | 水平右移 |
| `the camera tilts down` | 镜头下倾 |
| `the camera pans to the left` | 镜头左摇 |
| `the camera pans to the right` | 镜头右摇 |
| `the camera tilts up` | 镜头上倾 |
| `push in to close-up` | 推近至特写 |
| `pull out to wide shot` | 拉远至全景 |
| `dolly in` | 轨道推进 |
| `dolly out` | 轨道拉远 |
| `tracking shot` | 跟随镜头 |
| `orbit shot` | 环绕镜头 |
| `crane shot up` | 摇臂上升 |
| `crane shot down` | 摇臂下降 |

## 3. 景别

| 术语 | 说明 |
|------|------|
| `extreme close-up` | 极特写 |
| `close-up` | 特写 |
| `medium close-up` | 近景 |
| `medium shot` | 中景 |
| `full shot` | 全身景 |
| `wide shot` | 广角景 |
| `extreme wide shot` | 极广角 |
| `establishing shot` | 开场镜头 |
