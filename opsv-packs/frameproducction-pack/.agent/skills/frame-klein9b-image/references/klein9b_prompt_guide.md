# Klein9B Prompt Guide — 提示词编写指南

## 1. 基本结构

```
[场景/道具描述] + [光影描述] + [风格标签]
```

## 2. 场景 Prompt 模板

```
{场景名称}主馆内景。{建筑风格}，{材质描述}，{光影描述}。
镜头位于{视角位置}，{构图描述}。
{风格标签}，{画质标签}。
```

### 示例

```
杭州国家版本馆主馆内景。国家版本馆于2022年建成，王澍设计，以"文献中心"为设计理念，建筑外墙为耐候钢板与回收砖瓦砌筑，内部空间以"当代宋韵"为核心审美。镜头位于版本馆中庭，仰视一道高约八米、宽约十二米的龙泉青瓷屏风。屏风由数十块独立青瓷板拼合而成，每块约一米乘两米，板面垂直悬挂于钢结构框架上，板间缝隙约五毫米。青瓷釉面为经典龙泉粉青色，这是宋代龙泉窑最具代表性的釉色——一种介于青绿与米白之间的微妙色调，釉面呈现一种柔和的"玉石感"。
Chinese architectural photography, 8K detail, cinematic lighting.
```

## 3. 道具 Prompt 模板

```
{道具名称}，{材质}，{尺寸}，{颜色}，{装饰细节}。
{背景描述}，{光影描述}。
{风格标签}。
```

## 5. 注意事项

1. **prompt 必须 ≥ 50 字符** — 否则 validate 警告
2. **CFG = 1 是 Flux 默认值** — 不要随意修改
3. **steps = 6** — Klein9B 是 distilled 模型，6 步足够
4. **seed = -1 随机** — 交付前固定 seed
5. **分辨率 1920x1088** — Klein9B 的最佳分辨率
