# Voice Spec Guide — 声音设计描述编写指南

## 1. 设计字段完整列表

编写声音设计描述时，必须尽可能覆盖以下字段：

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| `性别` | ✅ | 男性/女性/中性 | `男性` |
| `年龄` | ✅ | 年龄段 | `25-35 岁` |
| `口音` | | 方言/地域特征 | `标准普通话` |
| `音高` | | 音调高低变化 | `中调起音，随情绪递进逐步拔高` |
| `语速` | | 说话快慢节奏 | `初始平缓，后段急促` |
| `音量` | | 响度层次 | `从轻柔克制逐步提升至大喊` |
| `音色质感` | | 声音质地 | `略带憨厚的普通男声，混有湿润感` |
| `清晰度` | | 吐字清晰度 | `初始清晰，崩溃后含混` |
| `情绪` | ✅ | 情感曲线 | `祝福→不舍→绝望` |
| `语调` | | 说话语气 | `平和→高亢→下沉哭腔` |
| `语气` | | 态度倾向 | `故作洒脱→无法克制→绝望无助` |

## 2. 编写技巧

### 情绪弧线设计
不要只写单一情绪，要描述**情绪的变化过程**：
```
错误：情绪：悲伤
正确：情绪：强装镇定的祝福→不舍的急切呼唤→撕心裂肺的绝望哀求
```

### 声音质感描述
使用**通感比喻**帮助模型理解：
```
"略带憨厚的普通男声，混有雨中的湿润感，情绪爆发时嗓音沙哑哽咽"
```

### 节奏变化
描述**语速和音量的动态变化**：
```
"初始平缓故作镇定，后段急促慌乱，重复呼喊时几乎无停顿"
```

## 3. 常用角色音色预设

### 旁白（Narrator）
```
性别：男性 | 年龄：30-40 | 口音：标准普通话
音高：中调 | 语速：平稳 | 情绪：客观冷静
音色质感：沉稳磁性，有厚度
```

### 青年主角（Young Hero）
```
性别：男性 | 年龄：18-25 | 口音：标准普通话
音高：中偏高 | 语速：适中偏快 | 情绪：坚定有力
音色质感：清亮有活力
```

### 反派（Villain）
```
性别：男性 | 年龄：35-45 | 口音：略带南方口音
音高：偏低 | 语速：缓慢 | 情绪：阴冷从容
音色质感：低沉沙哑，有压迫感
```
