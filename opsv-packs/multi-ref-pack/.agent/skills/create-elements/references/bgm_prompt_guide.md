# BGM Prompt 编写指南

> 适用阶段：S7 声音设计

---

## Prompt 结构

```
[风格], [乐器], [节奏], [情绪], [氛围].

示例：
Traditional Chinese instrumental piece. Guzheng melody
with bamboo flute harmony. Starts slow and peaceful,
gradually builds tension with soft taiko drum.
Ambient temple atmosphere. Morning.
Suitable for BGM generation.
```

## 风格速查

| 风格 | 适合场景 |
|------|---------|
| traditional Chinese | 古风、武侠 |
| orchestral epic | 史诗、战斗 |
| ambient | 环境、沉思 |
| electronic | 科幻、现代 |
| jazz | 都市、夜晚 |
| folk acoustic | 温馨、日常 |
| horror dark | 恐怖、悬疑 |

## 乐器表

| 乐器 | 适合风格 |
|------|---------|
| guzheng | 中国风 |
| bamboo flute | 中国风、冥想 |
| erhu | 中国风、悲伤 |
| taiko | 中国风、战斗、史诗 |
| piano | 通用 |
| strings | 史诗、浪漫 |
| synth pad | 电子、科幻 |
| guitar | 民谣、都市 |

## 节奏

| 节奏 | 说明 |
|------|------|
| slow | 舒缓、渲染 |
| moderate | 中性进行 |
| fast | 紧张、兴奋 |
| building | 渐强（从弱到强） |
| pulsating | 脉冲感（惊悚） |

## 情绪映射

| 场景 mood | 推荐 BGM 情绪 |
|-----------|-------------|
| tense | tense-building, pulsating |
| peaceful | calm, ambient, slow |
| epic | orchestral, building, grand |
| sad | melancholy, slow, strings |
| joyful | bright, upbeat, major key |
| mysterious | ambient, dark, minimal |
