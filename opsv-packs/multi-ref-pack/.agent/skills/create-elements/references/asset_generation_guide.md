# 资产生成指南

> 适用阶段：S4 资产生成

---

## 模型选择

| 资产类型 | 推荐模型 | 备注 |
|---------|---------|------|
| 角色定稿 | seedance | 一致性优先 |
| 场景概念 | seedance | 氛围优先 |
| 道具 | seedance | 细节优先 |

## 分辨率

| 资产类型 | 推荐分辨率 |
|---------|-----------|
| 角色（全身） | 1024×1024 |
| 角色（半身） | 1024×1024 |
| 场景（wide） | 1344×768 |
| 道具 | 768×768 |

## 生成优先级

1. **角色优先** — 角色是最高频复用资产，所有下游依赖角色图
2. **场景次之** — 场景定稿图，S6 视频生成需要
3. **道具最后** — 仅关键道具

## 角色变体清单

| 变体 | 说明 | 用途 |
|------|------|------|
| portrait | 正面半身 | S5 分镜角色引用 |
| full-body | 全身 | S6 视频姿势参考 |
| profile | 侧面 | 多视角一致性 |
| expression-{emotion} | 表情变体 | 情绪镜头 |

## 技术约束

- 不要生成带文字的图像（模型不擅长文字）
- 避免复杂多人场景（容易畸变）
- 角色面部占画面比例适中（过小面部细节丢失）
