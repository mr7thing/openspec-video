# 图像质量检查清单

> 适用阶段：S4 资产生成，S5 分镜草图，S6 视频生产

---

## S4 资产审阅标准（角色/场景/道具）

| 检查项 | 标准 | 通过/不通过 |
|--------|------|-----------|
| 角色面部一致性 | 与定档文档描述一致 | |
| 风格统一 | 与 project.md 定义风格一致 | |
| 无畸变 | 面部/手部无变形 | |
| 构图合理 | 主题居中，留白得当 | |
| 色彩准确 | 符合 color_palette 描述 | |
| 无文字 | 图像中不应出现文字 | |

## S5 分镜草图审阅标准

| 检查项 | 标准 |
|--------|------|
| 构图层级 | 角色位置关系清晰 |
| 时间线连贯 | 9 格构成从起→承→转→合的完整过程 |
| 景别正确 | 与 Script.md 定义的 shot_type 一致 |
| 运镜可读 | 看草图能理解镜头运动方向 |
| 角色参考 | 角色姿势与角色定稿一致 |

## S6 视频审阅标准

| 检查项 | 标准 |
|--------|------|
| 角色一致性 | 视频中角色外观与定稿图一致 |
| 画面流畅度 | 无跳跃、卡顿 |
| 构图符合分镜 | 与 storyboard 草图的构图一致 |
| 无闪烁 | 画面不闪烁/抖动 |
| 无畸变 | 角色/场景在运动中不变形 |

## 不通过处理

不合标准 → `opsv iterate <task_path>` → 修改 prompt/参数 → 重新执行
