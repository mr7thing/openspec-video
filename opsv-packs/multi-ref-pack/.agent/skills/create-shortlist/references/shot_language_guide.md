# 镜头语言指导

> 适用阶段：S3 Shot 场记 — 为每个 Shot 决定摄影参数

---

## 景别分类

| 景别 | 英文 | 说明 | 用途 |
|------|------|------|------|
| 大远景 | extreme-wide | 人物极小，环境为主 | 建立场景 |
| 远景 | wide | 全身，环境占主体 | 交待位置关系 |
| 中景 | medium | 膝上/腰上 | 对话、动作 |
| 近景 | close-up | 胸上/肩部 | 表情、情绪 |
| 特写 | extreme-close-up | 局部（眼/手） | 细节、焦点 |
| 主观视角 | POV | 角色所见 | 代入感 |
| 过肩 | over-shoulder | 两人对话 | 正反打 |

## 机位

| 机位 | 说明 |
|------|------|
| eye-level | 平视，中性 |
| low-angle | 仰视，角色显强势 |
| high-angle | 俯视，角色显弱势 |
| bird's-eye | 俯视全景，上帝视角 |
| over-shoulder | 过肩，对话常用 |

## 运镜

| 运镜 | 说明 |
|------|------|
| static | 固定机位 |
| pan | 左右摇 |
| tilt | 上下摇 |
| dolly | 推拉（物理移动） |
| crane | 升降 |
| handheld | 手持晃动（纪实感） |
| tracking | 跟拍 |

## 景别组合（转场用）

| 组合 | 效果 |
|------|------|
| wide → close-up | 快速聚焦 |
| close-up → wide | 揭示环境 |
| medium → medium | 对话节奏 |
