# Seedance 参数手册

> 适用阶段：S6 视频生产

---

## 参数速查

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| duration | 4 | 1-10 | 视频时长（秒），4~6s 最佳 |
| seed | -1 | -1/任意整数 | -1=随机，固定 seed 可复现 |
| fps | 24 | 24/30 | 帧率 |
| cfg_scale | 7 | 1-30 | 提示词遵循度 |
| steps | 25 | 10-50 | 采样步数 |

## Duration 建议

| 用途 | 推荐 duration |
|------|-------------|
| 短动作（转头/表情） | 2-3s |
| 标准镜头 | 4-5s |
| 运镜（dolly/pan） | 4-6s |
| 长镜头叙事 | 6-8s |
| 超过 8s | 不推荐（质量下降） |

## Seed 策略

| 场景 | seed |
|------|------|
| 初代生成 | -1（随机） |
| 迭代修正 | 固定 seed（保持构图一致，只改 prompt） |
| 最终交付 | 固定 seed（可复现） |

## 注意事项

1. **不支持 frame refs** — Seedance 目前没有前后帧参考机制
2. **多参考图** — 角色 + 场景 + 分镜草图 作为独立 image refs 传入
3. **seed = -1 每次不同** — 定稿前固定 seed
4. **fps 字段 OPSV 可能不接受** — 检查 segment schema 是否支持
