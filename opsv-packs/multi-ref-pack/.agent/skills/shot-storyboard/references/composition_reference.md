## 三色运动标注规范

### 红色线（人物/物体运动）— 实线箭头

| 线型 | 含义 | 使用场景 |
|------|------|---------|
| → 水平实线箭头 | 水平移动 | 行走、奔跑、滑步 |
| ↗ 斜向实线箭头 | 斜向移动 | 跳跃、冲刺、飞踢 |
| ↓ 垂直实线箭头 | 垂直移动 | 坠落、跃起、下蹲 |
| ⟳ 弧形实线箭头 | 弧线运动 | 挥砍、旋转、绕圈 |
| ⇢ 虚线实箭头 | 快速运动 | 冲刺、闪现、瞬移 |
| ··· 点状轨迹 | 物体飞行路径 | 飞刀、子弹、箭矢 |

### 蓝色线（镜头运动）— 虚线箭头

| 线型 | 含义 | 使用场景 |
|------|------|---------|
| → 水平虚线箭头 | 跟移/横摇 | tracking shot, pan |
| ↗ 斜向虚线箭头 | 推近/拉远 | dolly in/out |
| ↑ 垂直虚线箭头 | 升降 | tilt up/down, crane |
| ⟳ 弧形虚线箭头 | 环绕 | arc shot, orbit |
| ⊘ 固定标记 | 固定机位 | static camera |

### 绿色线（特效/冲击/能量）— 波浪线/点线箭头

| 线型 | 含义 | 使用场景 |
|------|------|---------|
| → 水平波浪箭头 | 水平冲击波/气浪 | 爆炸冲击、拳风 |
| ↗ 斜向波浪箭头 | 斜向能量束 | 光波、魔法弹道 |
| ◎ 同心圆波浪 | 环形冲击扩散 | 落地冲击波、声波 |
| ↑ 垂直波浪箭头 | 上升能量柱 | 变身光柱、能量爆发 |
| ✦ 放射状虚线 | 闪光/爆裂 | 枪口火花、爆炸闪光 |
| ≋ 波动虚线 | 持续能量场 | 防护罩、力场波动 |

## 九宫格运动节奏

```
[格1] 松               [格2] 起               [格3] 承
 ⚫灰黑铅笔稿           ⚫灰黑铅笔稿           ⚫灰黑铅笔稿
 🔵固定                  🔴入画→               🔴加速→→
 🔴无                    🔵跟摇→               🔵跟摇→→
 🟢无                    🟢无                  🟢无

[格4] 转               [格5] 高潮              [格6] 反应
 ⚫灰黑铅笔稿           ⚫灰黑铅笔稿           ⚫灰黑铅笔稿
 🔵切中景               🔴挥砍↓                🔴后退←
 🔴转身⟳               🔵慢镜头               🔵静止⊘
 🟢无                    🟢◎冲击波扩散          🟢✦火星飞溅

[格7] 再起             [格8] 极                  [格9] 合
 ⚫灰黑铅笔稿           ⚫灰黑铅笔稿           ⚫灰黑铅笔稿
 🔵切全景               🔴冲刺↗                🔴收势
 🔴入画→→               🔵急速推↗              🔵固定⊘
 🟢无                    🟢↑能量光柱             🟢→光线消散
```

**运动节奏原则**：
- 格1-3：建立空间 + 启动运动（缓起，绿线通常不出现）
- 格4-6：冲突 & 高潮（红线+绿线密度最高）
- 格7-9：收束 + 定格（运动归零，绿线消散）

## 镜头运动全表

### 基础运动

| 运动 | 英文 | 🔵蓝线 | 视觉效果 | 情感用途 |
|------|------|--------|---------|---------|
| 固定 | Static | ⊘ | 画面不动，靠内容驱动 | 客观、观察、稳定 |

## 正反打体系

正反打（Shot / Reverse Shot）是对话场景的核心镜头语言。

### 基本结构

```
[建立镜头] → [A 的过肩镜头] → [B 的反打] → [反应镜头] → [双人镜]
Establishing → OTS A      → OTS B    → Reaction  → Two-shot
```

### 正反打变体

| 类型 | 描述 | 机位 | 用途 |
|------|------|------|------|
| 标准正反打 | A 过肩 → B 过肩 | 两人身后交替 | 日常对话 |
| 单人正反打 | A 单人中景 → B 单人中景 | 分别拍单人 | 情感对话、对峙 |
| 内反打 | 从 A 视角看 B（不带 A 的肩膀） | POV 机位 | 主观代入 |
| 外反打 | 越过 A 肩膀看 B | OTS | 客观旁观 |
| 荷兰角正反打 | 正反打都倾斜 | 倾斜±15° | 不安、失衡感 |
| 高低差正反打 | A 仰拍 / B 俯拍 | 一高一低 | 权力关系不对等 |

### 正反打注意事项

1. **轴线不跨** — A 始终在左，B 始终在右（180° 规则）
2. **景别对称** — 正反打的景别通常一致（对等关系）或刻意不一致（权力差）
3. **越轴要有动机** — 主动越轴 = 关系变化、立场转换

## 180° 轴线规则

```
         场景轴线（Action Line）
    A ●───────────────────────● B
               ↑
         180° 半圆工作区
         (摄像机只能在这边)
```

| 概念 | 说明 |
|------|------|
| 轴线 | 两个角色之间的连线，或角色视线方向 |
| 工作半圆 | 摄像机只能在轴线一侧 180° 范围内活动 |
| 越轴 | 摄像机跨过轴线到另一侧 → 人物位置关系颠倒 |
| 合理越轴 | 用中性镜头（骑轴/正面）过渡后再越轴 |
| 假越轴 | 角色走动改变轴线，摄像机不动 → 新轴线自然建立 |

## 剪辑衔接语法

| 技巧 | 说明 | 九宫格中的体现 |
|------|------|--------------|
| 动接动 | 上个镜头有运动，下个镜头接同一运动方向 | 格2 红线→ / 格3 红线→ 同向 |
| 视线匹配 | A 看向画右 → 切 B 在画左回应 | 格4/格5 正反打 |
| 动作匹配 | 上个镜头结束动作 = 下个镜头起始动作 | 格3 收拳 / 格4 拳击中 |
| 30° 规则 | 同主体两个镜头角度差 ≥30° | 格4→格5 机位明显变化 |
| 景别跳跃 | 同主体相邻镜头景别差 ≥2 档 | wide→close-up，不 wide→medium |

## 经典镜头句式

### 句式一：建立-正反打-反应（对话场景）

```
格1: [wide] 两人位置关系，建立空间           🔵固定
格2: [medium OTS] A 说话                      🔵固定
格3: [medium OTS] B 回应（正反打）            🔵固定
格4: [close-up] B 听到后的微表情（反应镜头）  🔵慢推↗
```

### 句式二：全景-中景-特写（揭示）

```
格1: [extreme-wide] 大环境 → 🔵缓推↗
格2: [wide] 角色进入视野 → 🔵推↗
格3: [medium] 角色发现什么 → 🔵停 ⊘
格4: [close-up] 角色震惊表情 → 🔵急速推↗
```

### 句式三：特写-拉远-全景（退场）

```
格1: [close-up] 角色面部 → 🔵慢拉←
格2: [medium] 角色转身 → 🔵继续拉←
格3: [long] 角色走远 → 🔵拉←
格4: [extreme-wide] 角色消失在地平线 → 🔵停⊘
```

### 句式四：动作-反应-结果（战斗）

```
格1: [medium] A 挥拳 → 🔴弧线挥出 → 🔵固定 → 🟢无
格2: [medium] B 中拳 → 🔴后退 ← 🔵跟推→ → 🟢◎冲击扩散
格3: [close-up] B 痛苦表情 → 🔴微颤 → 🔵固定 → 🟢✦火星溅出
格4: [wide] B 倒地 → 🔴↓坠落 → 🔵俯拍↓ → 🟢尘烟→扩散
```

### 句式五：越轴转场（关系变化）

```
格1: [medium] A 在左 B 在右（标准对话） → 🔵固定
格2: [medium] 沉默、A 转身走向镜头（骑轴） → 🔴走向镜头 → 🔵固定
格3: [medium] 新轴线：B 在左 A 在右 → 🔵切新机位
格4: [close-up] B 获得主动权 → 🔵仰拍↑
```

## 景别衔接规则

| 前镜 → 后镜 | 效果 | 注意事项 |
|------------|------|---------|
| wide → medium | 渐进聚焦 | 安全选择 |
| wide → close-up | 戏剧跳跃 | 必须 ≥2 档 |
| close-up → wide | 揭示环境 | 反差冲击 |
| medium → medium | 平稳过渡 | 角度差 ≥30° |
| same as before | 跳切（Jump Cut） | 刻意为之才有效 |
